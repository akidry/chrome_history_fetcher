# Chrome History Fetcher

Chrome閲覧履歴を取得・分析するためのツールです。

## 機能

- Chromeの閲覧履歴データベースからデータを抽出
- 訪問したWebサイトの統計情報を生成
- 閲覧パターンの視覚化

## 使い方

1. リポジトリをクローン
2. 必要なパッケージをインストール
3. スクリプトを実行

```bash
git clone https://github.com/akidry/chrome_history_fetcher.git
cd chrome_history_fetcher
# その他の手順...
```

## 必要条件

- Python 3.8+
- その他の依存パッケージ
